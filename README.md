# Healthcare Appointment Analytics Dashboard 📊

## 📌 Project Overview
This project focuses on analyzing healthcare appointment data using Tableau to derive meaningful insights about appointment trends, status distribution, and overall performance.

---

## 📊 Key Insights
- Total Appointments: 200
- No-show appointments are highest
- Monthly trends show fluctuations in patient visits
- Cancellation and scheduling patterns analyzed

---

## 📈 Dashboard Features
- KPI: Total Appointments
- Appointment Status Analysis (Bar Chart)
- Monthly Trend Analysis (Line Chart)

---

## 🛠 Tools Used
- Tableau
- Excel / CSV Dataset
